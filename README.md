# background-for-category

## Логика назначения фонов

| Тип страницы | Фон |
|---|---|
| Главная (`is_front_page`, `is_home`) | Изображение из строки «Главная страница» |
| Пост | Изображение рубрики верхнего уровня поста |
| Архив рубрики | Изображение этой рубрики (или её родителя верхнего уровня) |
| Остальные страницы | Только дефолтный цвет фона |

Рубрика верхнего уровня определяется функцией `get_top_term()`.  
Если изображение для страницы не задано — выводится только цвет фона из настроек.
