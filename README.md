# background-for-category

## Известные ограничения

**Страницы (page) не поддерживаются.**
В стандартном WordPress таксономия `category` не зарегистрирована для типа записи `page`, поэтому страницы всегда получают только дефолтный цвет фона без изображения. Поддержка появится автоматически, если сторонний плагин выполнит `register_taxonomy_for_object_type('category', 'page')`.

**Основная рубрика (primary category) не учитывается.**
Если посту назначено несколько рубрик, `get_top_term()` всегда берёт ту, у которой меньше `term_id` (т.е. созданную раньше). Настройка «основной рубрики» из Yoast SEO (`_yoast_wpseo_primary_category`) и RankMath (`rank_math_primary_category`) игнорируется.
