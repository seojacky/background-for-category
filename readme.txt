Устанавливает фоновые изображения для рубрик верхнего уровня WordPress.

== Description ==

WP Booster: Background for Category — плагин для автоматической установки фонового изображения страницы в зависимости от рубрики верхнего уровня.

Изображения выбираются прямо из медиабиблиотеки WordPress — никаких ручных загрузок файлов по FTP не требуется.

**Требования:**
* Функция или плагин `get_top_term()` должны быть установлены отдельно.

**Возможности:**
* Выбор фонового изображения из медиабиблиотеки для каждой рубрики верхнего уровня.
* Отдельный фон для главной страницы.
* Настраиваемый дефолтный цвет фона (применяется если изображение не задано).
* Страница настроек в меню «WP Booster» админ-панели.

== Installation ==

1. Загрузите папку плагина в `/wp-content/plugins/`.
2. Активируйте плагин через меню «Плагины» в WordPress.
3. Убедитесь, что функция `get_top_term()` доступна на вашем сайте.
4. Перейдите в «WP Booster → Фон категории».
5. Для каждой рубрики и главной страницы нажмите «Выбрать» и выберите изображение из медиабиблиотеки.
6. Укажите дефолтный цвет фона и сохраните настройки.

== Frequently Asked Questions ==

= Как назначить фон рубрике? =
Перейдите в «WP Booster → Фон категории», найдите нужную рубрику в таблице и нажмите кнопку «Выбрать» — откроется стандартная медиабиблиотека WordPress.

= Что такое get_top_term()? =
Пользовательская функция, которая возвращает рубрику верхнего уровня для текущего поста. Должна быть добавлена в `functions.php` вашей темы или через отдельный плагин.

= Что происходит, если изображение для рубрики не найдено? =
Применяется только дефолтный цвет фона, указанный в настройках плагина.
